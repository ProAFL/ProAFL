## 结论
- 所有 8 个特征对区分 error / correct 都达到统计显著（p ≈ 0）。
- 效应量 r 在 0.47–0.57 之间（按 Cohen 标准属于中到大效应），整体区分能力**明显弱于** VOC2012 / KITTI_8（两者 r 均 > 0.7）。
- 量化排序：**全程均值（iou_mean / conf_mean）> 后期均值 ≈ 起量延迟 > 早期均值**。
- 与前两个数据集一致：iou 系列略优于 conf 系列；early 系列最弱。
- VisDrone 难度更高，可能因目标小、密集，训练动态对 error 的辨识度下降。
