## 结论
- 所有 8 个特征对区分 error / correct 都达到统计显著（p ≈ 0）。
- 效应量 r 全部 > 0.7（按 Cohen 标准属于大效应），区分能力都很强。
- 量化排序：**全程均值（iou_mean / conf_mean）> 后期均值 > 起量延迟（D_iou / D_conf）> 早期均值**。
- 与 VOC2012 结果一致：iou 系列略优于 conf 系列，全程均值最佳。
- 与 VOC2012 不同：在 KITTI_8 上后期均值优于起量延迟，而 VOC2012 上起量延迟优于后期均值。
